# Getting Started
Installing the dependencies

	
    npm install or yarn install

Setting the Environment Variable

    touch .env
    nano .env
	#.env
	PORT =  {port}
	BASE_URL = {baseurl}
	DB_URL = {databaseurl}

Building the Project

    npm build or yarn build

Running the Project

	
	
	#dev mode
    npm dev or yarn dev
    #production mode
    npm start or yarn start
